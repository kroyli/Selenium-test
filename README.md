# Selenium Test Project

## About the Project
This is a simple Java project using Selenium WebDriver.  
The program opens the Google Chrome browser, navigates to [https://www.google.com](https://www.google.com), and then closes the browser.

## Project Requirements
- Java JDK 11 or higher
- Installed Google Chrome browser
- ChromeDriver (version must match the installed Chrome)
- Maven (if used to build the project)

## Setup Instructions

1. **Download ChromeDriver**  
   Download from the official site: [ChromeDriver](https://sites.google.com/chromium.org/driver)

2. **Add ChromeDriver to Project or PATH**  
   - Place ChromeDriver in your project folder, **or**  
   - Add it to your system PATH.

3. **Clone the Repository**  

git clone https://github.com/kroyli/selenium-test.git
cd selenium-test
